## 针对特定网站的数据爬取和收集

**文件名称：pubmed.py**

**文件描述：**

受女朋友之托，需要针对专业文献网站pubmed的搜索结果进行爬取操作，该py文件可以实现的功能：
1. 利用python+selenium，pubmed网站模仿人类操作
2. 支持发送邮件提醒
3. 将搜索结果展示的文章的标题、摘要、发表时间、PMID等文章信息导出到csv文件中，方便排查（并不下载文章本身，仅自动收集文章信息，禁止违法行为哦～）

该文件demo适用于所有针对于特定网站的数据收集
